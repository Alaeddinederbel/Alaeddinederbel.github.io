Personal Projects

CAN BUS + STM32

CAN Dashboard — STM32 Multi-ECU System

Three simulated ECUs communicate over a shared CAN bus on an STM32F429ZIT6. A dashboard receiver decodes and displays real-time vehicle data over UART. Includes a Python software simulation — byte-for-byte compatible with the C firmware — that validates the full protocol on any PC without hardware.

ECUx3→CAN Bus→Dashboard→UART
STM32F429 CAN Bus C/C++ Python Simulation UART
OTA + IoT

OTA Firmware Update System — ESP32 + ESP8266

Production-grade Over-The-Air firmware update system for IoT devices. The ESP32 acts as trigger master — it sends a single HTTP request to wake the ESP8266, which autonomously downloads a new firmware binary from a local server, flashes itself, and reboots. No cables. No manual visits. Just push and reboot.

ESP32 trigger→HTTP→ESP8266→Flash & Reboot
ESP32 ESP8266 OTA Wi-Fi / HTTP C/C++
VISION + ML

ESP32-CAM Person Detection & Tracking

Real-time person detection and servo tracking on a single ESP32-CAM chip. OV2640 captures frames, TFLite inference detects persons, a PID controller drives an SG90 servo to track movement, and telemetry is streamed live via MQTT over Wi-Fi to a dashboard.

OV2640→TFLite→PID→SG90
ESP32-CAM MQTT PID Controller C++
SECURITY + PIC18F

Secure Digital Lock System (PIC18F)

Embedded access control system on a PIC18F microcontroller with PIN authentication and a keypad-based user interface. Features secure code verification, lockout after failed attempts, LCD status feedback, and relay-driven lock actuation.

Keypad→PIC18F→Auth→Relay
PIC18F C Keypad LCD Access Control
